# 💼 Max's Portfolio Website

A personal portfolio website built with vanilla HTML, CSS, and JavaScript — showcasing my skills, education, and contact information as a Frontend Developer.

🔗 **Live Demo:** [Coming Soon]

---

## 📸 Preview

![Portfolio Preview](main.png)

---

## 🚀 Features

- Responsive design for all screen sizes
- Typing animation on the hero section
- Spinning glow ring around profile image
- Animated skill progress bars (scroll-triggered)
- Education timeline
- Working contact form (EmailJS integration)
- Smooth scroll with active nav link highlighting
- Mobile slide-in navigation drawer

---

## 🛠️ Built With

- HTML5
- CSS3 (custom properties, animations, flexbox, grid)
- Vanilla JavaScript
- [Font Awesome 6.5](https://fontawesome.com/) — icons
- [Google Fonts – Poppins](https://fonts.google.com/specimen/Poppins)
- [EmailJS](https://www.emailjs.com/) — contact form email delivery

## 🧑‍💻 Getting Started

To run locally, just clone the repo and open `index.html`:

```bash
git clone https://github.com/Maxorleans55/Portfolio-Application-.git
cd Portfolio-Application-
```

Then open `index.html` in your browser — no build tools or installs needed.
